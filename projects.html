Equitable Recruitment Analysis

Developed an advanced data visualization solution to analyze resume data, enabling a comprehensive evaluation of candidate demographics, education, IT experience, and location. Utilized SQL and Python for data integration and extraction from AI-powered application tracking systems (ATS). Leveraged Power BI for data segmentation and insights, supporting strategic recruitment decisions with a focus on veterans, veteran spouses, and individuals with disabilities. Designed scalable databases and an intuitive front-end interface for efficient data querying, enhancing diversity-focused talent acquisition.

The Veteran Resume Analysis Page serves as the landing page of the Decision Support Dashboard, summarizing all the information related to the data available in the database.

The Experience Analysis Page provides visuals of top applicants across different sectors, analyzing experience by state and the skills applicants possess. This helps answer business questions about the best applicants with specific experience levels by sector, state, and the skills they bring to the job.

The Applicant Profile Dashboard simplifies the search for applicants, allowing employers to closely review selected applicant information. Searches can be conducted using the applicant's name or job title. This dashboard also provides the option for employers to initiate contact via email or phone with the selected applicants.

The Location Analysis Dashboard utilizes ArcGIS to conduct a deeper analysis of applicant locations, identifying the cities and states where veterans, their families, and disabled applicants reside. Additionally, this dashboard helps visualize which cities or states have the most qualified applicants in specific sectors. This analysis enables employment-focused decision-making based on geographic insights and sector-specific applicant data.

The Resume Database Dashboard visualizes all the data in the database in a tabular format, allowing users to easily track applicants and their resume information. This page serves as a centralized view of the raw data, eliminating the need to return to the ATS to access the dataset.

  • The dashboards I created facilitated data-driven decision-making in recruitment and applicant management by providing clear insights into applicant demographics, experience, and skills. These visualizations enabled the team to quickly identify top candidates based on location, sector, and expertise, ensuring that the hiring process was more efficient and targeted.
  • The project faced difficulties extracting structured data from unstructured resumes due to varied formats and inconsistencies in presenting key details like education, experience, and skills. Advanced text extraction techniques and natural language processing methods were applied to convert this unstructured data into usable formats.
  • Streamlined the data pipeline process, minimizing the need for manual data entry by automating key workflows. New data in the database is automatically updated on the dashboard.
  • I used SQL queries to extract, join, and transform raw data from the ATS into structured formats for analysis. Additionally, Python (with libraries like Pandas) was utilized for data cleansing, validation, and advanced integration tasks to ensure consistency and accuracy across datasets.
